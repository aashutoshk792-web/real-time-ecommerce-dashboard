\# Real-Time E-Commerce Analytics Dashboard



A real-time e-commerce analytics dashboard built using FastAPI, React, Kafka, and MySQL.



\## Features



\* Real-time sales monitoring

\* Live event tracking

\* Revenue analytics

\* Product performance dashboard

\* Customer analytics

\* JWT Authentication

\* Interactive Charts

\* FastAPI Backend

\* React Frontend



\## Tech Stack



\### Frontend



\* React.js

\* Material UI

\* Recharts

\* Axios



\### Backend



\* FastAPI

\* Python



\### Database



\* MySQL



\### Streaming



\* Apache Kafka



\## Project Structure



backend/

frontend/

consumer/

producer/



\## Installation



\### Backend



```bash

pip install -r requirements.txt

uvicorn main:app --reload

```



\### Frontend



```bash

npm install

npm run dev

```
